Welcome to Contact Managemnt POC
   This application is built using .Net Core 2.2, It is using Repository Pattern it can use .net Entity framework core and ADO.net also, The Web project can be further exposed as serives.
   Custom Logging and exception handling can be further added for Administrative purpose.
   The Images are currently Hard coded but they must be moved to Network file system saved via encryption.
   User Registration and Contacts of the user must be seprated by applying composite key involving unique loginid and contactId.
   The stored procedures and schema are included in project as text files.
